  • Client: Start Movie Findr
  • Date: Nov 2016

The Partiest

A full stack application that lets you organize and create office party for your company. This application uses Mongo, Express, Angular and Node to create, read, update and delete your party posts. Nodemailer was used to send party invites to invitee's email. Angular Materialize was used for styling. Ng-map is an angular third-party API that allows google map to get direction to the party destination.

  • Client: Start The Partiest
  • Date: Dec 2016
  • Service: Github and Heroku
